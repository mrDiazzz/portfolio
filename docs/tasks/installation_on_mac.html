Installation on Mac

This task describes how to install the desktop version of the Talking Circle® messenging application.

Follow the steps to install the software on a Mac OS X platform.

In case you have an older version installed on your system, make sure you uninstall it first. Leave the settings in place, as they are used by the newer version.

  1. Download the installation kit Talking Circle Web Site
  2. Extract the archive
  3. Drag the application icon to your Applications folder
